Auswählen eines Themes

Publii bietet bereits Themes an, die man als Basis für seine Seite auswählen kann. Dazu klickt man auf Themes → Menü (oben rechts) → Themes. Dort kann man ein Theme zuerst unter folgendem Link downloaden und den ZIP-Ordner per “drag- and drop” hineinziehen.

Seite auf GitHub erstellen

1) Token erstellen Um eine Seite auf GitHub zu erstellen, muss man zunächst ein sog. “Zugangs-Token” erstellen. Damit kann Publii auf die Github-Seite zugreifen und Änderungen vornehmen, so dass es die Website erstellen und bearbeiten kann, sobald man in der Publii-App auf den Sync-Button klickt.
